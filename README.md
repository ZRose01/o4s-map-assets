# O4S Map Assets

Static data + images for the O4S Nebraska town-hall map, served via
GitHub Pages and fetched at serve time by the Squarespace embed
(`panel.html` in the websiteMap project).

- `events.json` — generated by `build_events.py` from the town-halls
  tracker CSV. Never edit by hand; re-run the script, then commit and
  push here.
- `counties.geojson` — Nebraska county boundaries (U.S. Census
  TIGERweb). Static.
- `images/` — event photos. Filenames are referenced by the `image`
  field in events.json (the `Image` column in the tracker CSV).

Pages serves from `main` at the repo root; changes go live within
~10 minutes of pushing (CDN cache).

Do not add the tracker CSV or anything containing staff names or
personal phone numbers — this repo is public.
